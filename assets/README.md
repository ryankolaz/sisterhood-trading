## Format Notes

- **Photos:** Save as `.webp` for best performance. If you only have `.jpg` or `.png`, that works too — just rename the file to match the filename above (e.g., `hero-main.webp`). Or send me the originals and I'll convert them.
- **Video:** `.mp4` (H.264 codec). Keep under 15MB if possible. Compress with Handbrake if needed.
- **Dimensions:** The dimensions listed are the minimum recommended size. Larger is fine — images will be cropped to fit via CSS `object-fit: cover` (centers and fills the space). Smaller images will still work but may look soft.
- **Aspect ratios matter more than exact pixels.** Match the ratio and the image will look great at any resolution.
